# THY Route / Journal

THY ekosistemine entegre edilebilecek premium, interaktif seyahat planlama ve sosyal paylaşım platformu prototipi.

## 🌟 Özellikler
- **Premium Tasarım:** THY Kırmızısı ve Gece Mavisi ile "Glassmorphism" (cam/gölge) tabanlı UI.
- **İnteraktif Rota Planlama:** Gün bazlı planlama yapısı.
- **Çoklu Kullanıcı (Collab) Simülasyonu:** Arkadaşlarınızla aynı rotayı planlama deneyimi.
- **Mil Entegrasyonu:** Konaklama, etkinlik girişleri için "Millerle Öde" ve rota tamamlandığında "Mil Kazan" simülasyonu.
- **Dinamik Harita:** Gün değiştikçe otomatik güncellenen harita pinleri.

## 🚀 Kurulum (Local Test)

Proje Vanilla HTML/CSS/JS ile yazıldığı için doğrudan `index.html` dosyasını tarayıcıda açarak test edebilirsiniz. 
En iyi deneyim için VS Code üzerinden **Live Server** eklentisiyle çalıştırmanız önerilir.

Eğer Node.js yüklüyse:
```bash
npx serve .
```

## 🌍 Vercel'e Yükleme (Deployment)

Proje Vercel'e yüklenmeye tamamen hazırdır. Kök dizinde yer alan `vercel.json` sayesinde gerekli cache (önbellek) ayarları yapılandırılmıştır.

1. Vercel CLI yüklüyse terminalde `vercel` komutunu çalıştırın.
2. Veya GitHub reposuna yükleyip Vercel paneli üzerinden projeyi tek tıkla canlıya alabilirsiniz.
